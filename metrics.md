### pgmoneta_state

The state of pgmoneta

*   1 = Running

### pgmoneta_compression

The compression used

*   0 = None
*   1 = GZip
*   2 = ZSTD
*   3 = LZ4
*   4 = BZIP2

### pgmoneta_link

Use links to limit backup size

*   1 = Yes
*   0 = No
